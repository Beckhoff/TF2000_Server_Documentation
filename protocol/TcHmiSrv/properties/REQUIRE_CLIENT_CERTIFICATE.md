# A client certificate is required to establish an HTTPS connection

## General information

|  |  |
| - | - |
| Domain | `"TcHmiSrv"` |
| Full symbol path | `"TcHmiSrv.Config::REQUIRE_CLIENT_CERTIFICATE"` |
| Category | security |
| Visibility | AlwaysShow |
| Contained in every configuration by default | No |

```javascript
TcHmi.Server.readSymbol('TcHmiSrv.Config::REQUIRE_CLIENT_CERTIFICATE', data => {

        data.response.error ||

    // Handle result...
```
